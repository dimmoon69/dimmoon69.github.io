## <font color="red">Массивы</font>
Я решил, что логично будет изучить циклы в месте с массивами, так удастся лучше освоить их на практике.


* Синтаксис для создания нового массива – квадратные скобки со списком элементов внутри. В массиве могут храниться элементы разного типа. Нумерация элементов начинается с 0.

```js
var arr = [];//пустой
var strings = ["string","number","boolean",{text:'hello world',n:20}];

alert(strings[0]);//string
alert(strings[3].text);//hello world
alert(strings);//string,number,boolean,[object Object]
```

Также массив можно получить из строки с помощью функции split(разделитель)

```js
var nums = "12345"
var arr = nums.split(''); // [1,2,3,4,5]

var names = "Erich, Maria, Remarque";
var arr = names.split(', ');// ["Erich","Maria","Remarque"]
```

Другие полезные методы связанные с массивом мы рассмотрим позже. 

* Элементы массива можно изменять, удалять и добавлять новые

```js
var nums = [1,2,3,4];
//Допустим, наша задача изменить последний элемент,
//но если мы не знаем индекс последнего элемента, это становится немного затруднительно.
// для этого сделаем так:

//Чтобы узнать длину массива 
var len = nums.length; //4
nums[len-1] = 100; //теперь последний элемент массива равен 100

nums[0] = 100; //теперь 1 элемент массива равен 100 
```

* Добавление и удаление элементов:

```js
var nums = [1,2,3,4];

//Конец списка
var n = nums.pop();//удаляет последний элемент и возвращает его
alert(n);//4

nums.push(4);//добавит элемент 4 в конец массива

//Начало списка
var first = nums.shift();//1

nums.unshift(1); //добавит в начало массива 1

//также любой элемент можно удалить так
delete nums[1];
alert(nums);//1,null,3,4
```

К слову  push и unshift могут добавлять сразу несколько элементов, unshift(5,6)
